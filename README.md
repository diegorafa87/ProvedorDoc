# Gestão de Clientes - MERN Stack

Projeto inicial para um sistema de gestão de clientes em tempo real.

## Estrutura
- **backend/**: Node.js + Express
- **frontend/**: React.js
- **database**: MongoDB (a configurar)
- **upload de arquivos**: pronto para integração

## Como iniciar
1. Instale as dependências em cada pasta (`backend` e `frontend`).
2. Rode o backend: `npm start` na pasta backend.
3. Rode o frontend: `npm start` na pasta frontend.
